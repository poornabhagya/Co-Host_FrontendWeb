# Co-Host Ceylon - Premium Web Platform.

A high-end, fully responsive front-end web application built for **Co-Host Ceylon**, a luxury hospitality growth partner managing Sri Lanka's finest villas and boutique hotels. 

This project focuses on delivering a "Boutique Luxury" digital experience through minimal design, elegant typography, and seamless interactive elements inspired by tropical modernism.

##  Key Features

* **Interactive Custom Map:** A tailor-made, transparent map of Sri Lanka featuring dynamic, clickable hotspots for property locations.
* **Synchronized Property Slider:** Seamless navigation between properties using custom arrows, automatically syncing the active property card with the map's location pin.
* **Luxury UI/UX:** A clean, airy design language utilizing elegant typography (Playfair Display & Montserrat) and a sophisticated tropical color palette (Emerald Green & Beige).
* **Smart Navigation:** Fixed, glass-morphism navigation bar with smooth scrolling and a fully responsive mobile menu (includes body-scroll locking to prevent background scrolling).
* **Fully Responsive:** Flawless layout adaptation across all devices, ensuring perfect viewing from small mobile screens to ultra-wide desktop monitors.

##  Tech Stack

* **Framework:** React.js (via Vite)
* **Language:** TypeScript (`.tsx`)
* **Styling:** Tailwind CSS
* **Icons:** Lucide React
